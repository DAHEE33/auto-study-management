# 🚀 Project: Study-Sync (Auto-Settlement Engine)

> **"관리 리소스 Zero를 지향하는 스터디 운영 자동화 솔루션"**
> 본 프로젝트는 카카오톡 챗봇, OCR, Google Sheets API를 결합하여 출석 인증, 벌금 정산, 스토리지 관리를 자동화하는 **Serverless 정산 시스템**입니다.

---

## 1. 🏗 System Architecture
* **Backend**: Python 3.11 / FastAPI (Asynchronous)
* **Database**: Google Sheets API (Source of Truth / NoSQL Interface)
* **Storage**: Google Drive API (Image Hosting & Lifecycle)
* **OCR**: Google Cloud Vision API (Parsing Gurumi UI)
* **Infrastructure**: Railway (PaaS), GitHub Actions (Cron Jobs)

---

## 2. 📅 핵심 운영 및 시간 검증 로직

### 2.1 이중 시간 검증 (Dual-Time Validation)
* **제출 시각 (Server Time)**: 사용자가 사진을 전송한 시각 (익일 12:00 정오 마감).
* **공부 종료 시각 (OCR Time)**: 사진 내 타이머에 표시된 시각 (당일 01:00 이전 종료 필수).
* **로직**: `제출시각 < 익일 12:00` AND `종료시각 < 당일 01:00` 일 때만 정상 인증.

### 2.2 트랜잭션 유형
* **[일반인증]**: 사진 제출 → 이중 시간 검증 → 목표 시간 대조.
* **[반휴사용]**: 당일 목표 시간 **1h 고정** 및 주간 반휴 잔여량 0.5회 차감.
* **[주휴/특휴/월휴]**: 잔여량 1회 차감 및 즉시 PASS 처리 (사진 불필요).
* **[특휴-증빙]**: 병가/시험 등 사유 발생 시 사진 업로드 → 관리자 승인 대기(Pending).

---

## 3. 💰 벌금 및 정산 로직 (Zero-Sum Engine)

### 3.1 벌금 산정 (Penalty)
* **목표 미달**: 1h 이상 수행했으나 목표 미달 시 **-500원**.
* **심각 미달**: 1h 미만 수행 시 **-1,000원**.
* **부정/오류**: 날짜 불일치 또는 판독 불가 시 **-1,000원**.
* **결석**: 마감(정오) 시점까지 로그 부재 시 휴가 자동 차감 → 부족 시 **-2,000원**.

### 3.2 주간 1/N 정산 (Weekly Settlement)
* **실행**: 매주 월요일 12:05 (GitHub Actions).
* **로직**: 전체 벌금을 벌금 0원인 멤버($N$)로 배분 (100원 단위 절삭).
* **잔돈 처리**: 절삭 후 남은 잔돈은 해당 주 **누적 공부 시간 1위**에게 보너스 지급.

---

## ♻️ 4. 스토리지 관리 (Cleanup)
* **Cleanup Phase**: 매주 수요일 12:00 실행.
* **대상**: `createdTime < '당주 월요일 00:00'` 이미지 영구 삭제.
* **효과**: 구글 드라이브 무료 용량(15GB) 내에서 무한 운영 가능.

---

## 📊 5. Dashboard & Visualization (통계 및 대시보드)

### 5.1 실시간 통계 시트 (`Statistics_View`)
* **주간/월간 랭킹**: 멤버별 누적 공부 시간 및 목표 달성률 실시간 집계.
* **벌금 현황판**: 현재 적립된 전체 벌금액과 멤버별 예상 정산 금액(1/N 반영) 노출.
* **개인별 히트맵**: GitHub 잔디 심기 형태의 월간 출석 현황 시각화.

---

## 📱 6. User Interface (카카오톡 챗봇 UX)

### 6.1 사용자 인터랙션 Flow
* **스마트 메뉴 (Quick Reply)**:
    * `[🔥 오늘 인증]`: 사진 업로드 가이드 및 전송.
    * `[📈 내 현황]`: 현재 주간 누적 시간 및 남은 휴가 개수 즉시 응답.
    * `[📅 전체 랭킹]`: 통계 시트 웹뷰(Webview) 링크 제공.
* **인증 피드백**: OCR 판독 결과(시작/종료/순공 시간)를 요약하여 즉시 반환하며, 오류 발생 시 즉시 `[재촬영]` 버튼 노출.

---

## 🛠 7. Admin Control (운영 관리 로직)

### 7.1 관리자 전용 대시보드 (`Admin_Console`)
관리자는 Google Sheets를 GUI처럼 활용하여 시스템을 제어합니다.
* **특수 휴가(특휴) 승인**: 사용자가 증빙 사진을 올리면 `Daily_Log`에 `Pending` 기록 → 관리자가 시트의 체크박스 클릭 시 승인 알림 자동 발송.
* **멤버별 커스텀 설정**: `Member_Master` 시트에서 개인별 **목표 시간(m)** 및 **예치금**을 직접 수정 및 고정.
* **강제 조정**: 정산 오류 시 관리자가 시트의 벌금 액수를 직접 수정하면 차주 정산에 즉시 반영.
